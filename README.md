需要安装依赖包pyecharts，安装方法百度即可

打开PerConfig，修改pack_name为你需要测试的APP包名，如要测试UC浏览器即改为'com.UCMobile'，xxx_Activity为APP主活动，比如UC浏览器的主活动名称为'com.UCMobile/.main.UCMobile'，这里改为你自己的活动名称即可
